## 🌟 Overview
A powerful Python utility that scrapes thousands of free HTTP/SOCKS proxies from various public directories, concurrently tests them for latency and anonymity, and outputs a clean list of working IP addresses for your scraping tasks.

## ✨ Key Features
- 🕷️ **Multi-Source Scraping**: Gathers IPs from multiple public lists.
- ⚡ **Concurrent Testing**: Uses multithreading to test hundreds of IPs fast.
- 🛡️ **Anonymity Checks**: Verifies if the proxy leaks your real IP.

## 🚀 Getting Started

### Prerequisites
Make sure you have `Python` installed on your local machine.

### Installation
```bash
git clone https://github.com/HafezMosleh/proxy-scraper-tester.git
cd proxy-scraper-tester
pip install requests beautifulsoup4
```

### Usage
```bash
python3 main.py
```
