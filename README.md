# proxy-scraper-tester

Python script to scrape free proxies globally and test their latency/anonymity.

## Features
- 🚀 Fast and optimized
- 🔒 Secure by default
- 🛠 Easy to configure

## Installation
```bash
git clone https://github.com/HafezMosleh/proxy-scraper-tester.git
cd proxy-scraper-tester
```
